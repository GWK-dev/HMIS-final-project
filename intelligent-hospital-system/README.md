## AI for Software Engineering Final Project

### Project Overview
This project addresses low clinical attendance at Kiambu Level 5 Hospital through an AI-powered management system. Based on comprehensive research findings including patient satisfaction surveys and demographic analysis.

### Key Features
- **Predictive No-Show Analysis**: ML model predicting patient attendance
- **Digital Patient Records**: Solves manual file system issues
- **Smart Scheduling**: Reduces overcrowding (89.9% problem identified in research)
- **Analytics Dashboard**: Visualizes research insights and hospital metrics

### Research Foundation
Based on survey of 50 patients at Kiambu Level 5 Hospital:
- 89.9% reported clinic overcrowding
- 73% of respondents were female
- 48% aged 48+ years
- Key satisfaction dimensions analyzed: Physical Environment, Technical Quality, Interpersonal Relations, Communication, Accessibility

### Technology Stack
- **Backend**: FastAPI, SQLAlchemy, Scikit-learn
- **Frontend**: Streamlit
- **Database**: SQLite
- **ML**: Random Forest for predictions
